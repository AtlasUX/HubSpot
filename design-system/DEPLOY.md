# Deploying the design system gallery

The gallery app lives in `app/`. For Vercel (hs-designsystem project):

1. **Root Directory** → Leave empty (repo root)
2. **Install Command**: `cd HSP-Onboarding/design-system && npm install && cd app && npm install`
3. **Build Command**: `cd HSP-Onboarding/design-system/app && npm run build`
4. **Output Directory**: `HSP-Onboarding/design-system/app/dist`
